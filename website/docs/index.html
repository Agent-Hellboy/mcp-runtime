Current scope

MCP Runtime is still in active development, so this docs area focuses on the current runtime model rather than a polished end-to-end product guide. OIDC/JWT bearer-token validation is supported for mcp-sentinel services, but a full OAuth 2.1 authorization server, consent flow, and Dynamic Client Registration remain out of scope for this first cut.
